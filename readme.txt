模仿itnue的文章图片展示插件。
Show your post's picture,and look like Itune.

== Description ==

这是一个给懒人准备的插件!

如果你的一篇文章配有许多图片，而且你不需要太复杂的配置的话，这个插件一定能满足你。

默认情况下会在正文底部生成一个模仿itune的图片展示方式。

同时你也可以根据自己的需要使用代码标签在你的主题模版指定的位置进行展示，

目前使用的标签为 &lt;?php wp_piclist_tag(); ?&gt;。

When your post has more picture and you are Lazy people. 

I suggest you use the Plugin. you only do write and upload image file. The Plugin will be show this post's picture In the bottom of the post。

== Installation ==

1. 你可以通过使用后台或FTP将安装本插件到你的wordpress的wp-content/plugins目录中。
2. 在菜单中激活本插件
3. 在插件的配置页中选择你希望的展示方式。
4. 如果你打算使用标签来指定图片列表的展示位置，请将&lt;?php wp_piclsit_tag(); ?>标签放到你指定的模版文件中。

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can choose the image you need to show the way
4. If you intend to use the template tag, add &lt;?php wp_piclsit_tag(); ?&gt; placed on the need to display the template file.

For more help, please visit: http://webcaft.com/wp-piclist-plugin
